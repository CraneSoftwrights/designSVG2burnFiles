1. Burst a Crane SVG design file into individual SVG burn files - designSVG2burnFiles.xsl

Filename: designSVG2burnFiles.xsl

Import statement:

  • Common code supporting Crane's designSVG files - designSVG2burnFiles.xsl

This stylesheet is taking a collection of sets of layers and creating individual SVG files with only the layers in the given set. At the same time, any strokes that are magenta in colour are indicated to have a specified stroke width for the cutter to recognize as a cut, not an etch. A set of Inkscape action files is created and the invocation of Inkscape for these files.
