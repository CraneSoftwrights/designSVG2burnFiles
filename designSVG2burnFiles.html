1. Burst a Crane SVG design file into individual SVG burn files - designSVG2burnFiles.xsl

Filename: designSVG2burnFiles.xsl

This stylesheet is taking a collection of sets of layers and creating individual SVG files with only the layers in the given set. At the same time, any strokes that are magenta in colour are indicated to have a stroke width of 0.001in for the cutter to recognize as a cut instead of an etch. A set of Inkscape action files is created and the invocation of Inkscape for these files.

1.2. Convert all identified layers

c:build match="g[matches(@inkscape:label,':')]" use="'__all__', for $each in tokenize(@inkscape:label,'\s+')[matches(.,':')] return substring-before($each,':')" (xsl:key)

Find some layers that are building blocks

c:build match="g[starts-with(tokenize(@inkscape:label,'\s+')[2],'=')]" use="'__all__', tokenize(@inkscape:label,'\s+')[1]" (xsl:key)

Find more layers that are building blocks

c:assemble match="g[not(ancestor-or-self::*[contains(@style,'display:none')])] [tokenize(@inkscape:label,'\s+')[starts-with(.,'=')]]" use="'__all__',tokenize(@inkscape:label,'\s+')[1]" (xsl:key)
